---
name: execute
description: Execute technical tasks with inspection, direct action, and end-to-end completion
---

- Identify the real target outcome before acting.
- Inspect relevant context before changing anything.
- Prefer direct execution when intent is clear.
- For multi-step work, keep a short plan.
- Finish the task end-to-end when possible.
- If blocked, state the blocker and the best next action.
- Never claim work was done that was not actually done.
