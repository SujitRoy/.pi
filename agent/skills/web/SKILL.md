---
name: web
description: Static sites, blogs, and personal website development
---

- Static: Use generators (Hugo, Jekyll, Next.js) or handcraft HTML/CSS.
- Performance: Minify assets, optimize images, use CDN.
- Mobile-first: Responsive design, test on small screens.
- SEO: Semantic HTML, meta tags, sitemaps, structured data.
- Security: HTTPS, CSP headers, sanitize user content.
- Blog: Markdown with frontmatter, RSS feed, archive.
- Analytics: Privacy-focused (Plausible, Fathom) or self-hosted.
- Deployment: Git-based (GitHub Pages, Netlify, Vercel).
- Maintenance: Regular updates, broken link checks.
- Accessibility: Keyboard nav, alt text, contrast ratios.
